Projects

   Dollar Tree: Case Study - Designed three detailed dashboards on Sales, Cost and Macroeconomics analysis on Dollar Tree and Family Dollar stores across the United States using Tableau.Embedded 5 filters (Brand, Region, Time period and Category) to drill down hierarchy and filter analysis. Examined external environmental trends from an array of sources such as truck utilization, fuel prices, consumer price index, and their impact with retail and sales.
   Scalable COVID-19 analysis using Big Data - Constructed multiple distributed and scalable map-reduce jobs in Hadoop for a dataset with 10-million data points containing personID and coordinates of seat.Identified and displayed personIDs close contact with infected people using distance based computation using SparkSQL.Used SparkContext to retrieve data from the HDFS and implemented a clustering algorithm to find the number of clusters in the concert using PySpark MLlib.
   Movie Prediction Rating System, - Extracted 450,000 records from multiple sources in JSON format and merged them into a single holistic dataset. Preprocessed the data with the steps of feature engineering,Synthetic Minority Oversampling Technique (SMOTE), and PCA on the data.Implemented, Trained and Tested Support Vector Machine (SVM), Random Forest, and XGBoost using 3-fold Cross Validation in R.Attained a F1 score of 0.75 on the XGBoost as the best performing and 0.41 F1 score on SVM as least performing.
   BERT Question Similarity - Used a Bidirectional Encoder Representation from Transformers on question similarity on Quora Dataset Preprocessed a large dataset of 500,000 records by removing empty strings, lemmatization , truncating the text to get a proper dataset and feed It to the Encoder.Adding tokens and Padding elements to the data before giving to the BERT base model. After hyper parameter tuning, the model produced a high performance of F1 score as 0.8 value.
   Crypto currency Chatbot - Built a seq 2 seq model using LSTM with multi head attention model to create an interactive chatbot using corpus scrapped from Q&A’s from quora.The model achieved a bilingual evaluation understudy (BLEU) score of about 0.68
   Emotion recognition in Human Voice Audio - Extracted audio features such as Periodogram, Intensity, Fundamental Frequency, HNR from CREMA audio dataset containing over 7000 audio clips from various audio recordings using Parselmouth. Used GRU a RNN to predict the human emotions from the extracted audio features getting over 35% accuracy on testing data.
   NYC Green Taxi Tip Percent Prediction - Removed the missing data, feature engineering on few features and finding tip percentage for each ride for the Green Taxi Data for the year 2019. Multiple classification models to predict the tip percent of each ride taking the features such as location ID, trip time, payment type, passenger count, pickup time etc.
